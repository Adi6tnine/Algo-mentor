## 📋 Quick Deployment Steps

### 1️⃣ Prepare Your Repository

Your repository is already set up with all necessary configuration files:
- ✅ `render.yaml` - Render deployment configuration
- ✅ `netlify.toml` - Netlify deployment configuration
- ✅ `frontend/.env.production` - Production API URL
- ✅ All backend and frontend code ready

### 2️⃣ Deploy Backend to Render

**Step-by-Step:**

1. **Go to Render Dashboard**
   - Visit: https://dashboard.render.com
   - Sign up or log in

2. **Connect GitHub**
   - Click "New +" → "Web Service"
   - Click "Connect GitHub"
   - Authorize Render to access your repositories
   - Select: `Adi6tnine/Algo-mentor`

3. **Configure Service**
   - Render will auto-detect `render.yaml`
   - Click "Apply" to use the configuration
   - Or manually configure:
     - **Name**: `algomentor-backend`
     - **Branch**: `main`
     - **Build Command**: `mvn clean package -DskipTests`
     - **Start Command**: `java -Dserver.port=$PORT -jar target/algomentor-1.0.0.jar`

4. **Set Environment Variables**
   
   Render will auto-configure most variables from `render.yaml`, but verify:
   
   ```env
   SPRING_PROFILES_ACTIVE=prod
   JWT_SECRET=<auto-generated-by-render>
   JWT_EXPIRATION=18000000
   CORS_ALLOWED_ORIGINS=https://your-app.netlify.app
   RATE_LIMIT_REQUESTS=100
   RATE_LIMIT_DURATION=60000
   ```
   
   Database variables are auto-configured from PostgreSQL service.

5. **Deploy**
   - Click "Create Web Service"
   - Wait 5-10 minutes for deployment
   - Note your backend URL: `https://algomentor-backend.onrender.com`

6. **Verify Deployment**
   ```bash
   curl https://algomentor-backend.onrender.com/actuator/health
   ```
   Should return: `{"status":"UP"}`

### 3️⃣ Deploy Frontend to Netlify

**Step-by-Step:**

1. **Go to Netlify Dashboard**
   - Visit: https://app.netlify.com
   - Sign up or log in

2. **Import Project**
   - Click "Add new site" → "Import an existing project"
   - Choose "Deploy with GitHub"
   - Authorize Netlify
   - Select: `Adi6tnine/Algo-mentor`

3. **Configure Build Settings**
   
   Netlify will auto-detect `netlify.toml`, but verify:
   
   - **Base directory**: `frontend`
   - **Build command**: `npm run build`
   - **Publish directory**: `frontend/build`
   - **Branch**: `main`

4. **Environment Variables**
   
   Already configured in `netlify.toml`:
   ```env
   REACT_APP_API_URL=https://algomentor-backend.onrender.com/api/v1
   NODE_VERSION=18
   ```

5. **Deploy**
   - Click "Deploy site"
   - Wait 2-5 minutes
   - Note your frontend URL: `https://[random-name].netlify.app`

6. **Custom Domain (Optional)**
   - Go to Site Settings → Domain management
   - Click "Add custom domain"
   - Follow DNS configuration steps

### 4️⃣ Update CORS Configuration

**CRITICAL STEP:**

1. Go back to Render Dashboard
2. Navigate to your `algomentor-backend` service
3. Go to "Environment" tab
4. Update `CORS_ALLOWED_ORIGINS` with your Netlify URL:
   ```
   CORS_ALLOWED_ORIGINS=https://[your-netlify-url].netlify.app
   ```
5. Click "Save Changes"
6. Render will automatically redeploy

### 5️⃣ Test Your Deployment

1. **Visit your Netlify URL**
   - Should see the login page with modern dark UI

2. **Create a Test Account**
   - Click "Sign Up"
   - Fill in student details
   - Submit

3. **Login**
   - Use your credentials
   - Should redirect to dashboard

4. **Test Features**
   - View dashboard stats
   - Navigate to problems page
   - Try sync functionality
   - Check analytics

5. **Check Browser Console**
   - Should have no CORS errors
   - API calls should succeed

## 🔐 Security Configuration

### JWT Secret
Render auto-generates a secure JWT secret. To manually set:

```bash
# Generate secure secret
openssl rand -base64 64

# Add to Render environment variables
JWT_SECRET=<your-generated-secret>
```

### CORS Origins
Update after getting Netlify URL:

```env
# Single origin
CORS_ALLOWED_ORIGINS=https://your-app.netlify.app

# Multiple origins (with custom domain)
CORS_ALLOWED_ORIGINS=https://your-app.netlify.app,https://www.yourdomain.com
```

### Database Password
Render auto-generates secure database credentials. No action needed.

## 🐛 Troubleshooting

### Backend Issues

#### Build Fails on Render
**Symptoms:** Build fails with Maven errors

**Solutions:**
```bash
# Check Java version in render.yaml
env: java  # Should use Java 17+

# Check pom.xml dependencies
# Ensure all dependencies are available

# Check build logs in Render dashboard
# Look for specific error messages
```

#### Database Connection Failed
**Symptoms:** Backend starts but can't connect to database

**Solutions:**
1. Verify database is running in Render
2. Check DATABASE_URL environment variable
3. Ensure database and web service are in same region
4. Check database credentials

#### CORS Errors
**Symptoms:** Frontend can't connect to backend

**Solutions:**
1. Update CORS_ALLOWED_ORIGINS in Render
2. Include your Netlify URL
3. Redeploy backend
4. Clear browser cache

### Frontend Issues

#### Build Fails on Netlify
**Symptoms:** Build fails with npm errors

**Solutions:**
```bash
# Check Node version in netlify.toml
NODE_VERSION = "18"

# Check package.json dependencies
# Ensure all dependencies are listed

# Check build logs in Netlify dashboard
# Look for specific error messages
```

#### Can't Connect to Backend
**Symptoms:** API calls fail, CORS errors

**Solutions:**
1. Verify REACT_APP_API_URL in netlify.toml
2. Check backend is running on Render
3. Verify CORS is configured correctly
4. Check browser console for errors

#### Blank Page After Deploy
**Symptoms:** Site loads but shows blank page

**Solutions:**
1. Check browser console for errors
2. Verify build directory is correct (frontend/build)
3. Check React Router redirects in netlify.toml
4. Clear browser cache and reload

## 💰 Cost Breakdown

### Free Tier (Perfect for Testing)

**Render:**
- PostgreSQL: Free (1GB storage, 1GB RAM)
- Web Service: Free (512MB RAM, shared CPU)
- Limitations: Spins down after 15 min inactivity

**Netlify:**
- Hosting: Free
- Bandwidth: 100GB/month
- Build minutes: 300/month

**Total: $0/month**

### Production Tier (Recommended)

**Render:**
- PostgreSQL: $7/month (10GB storage, 1GB RAM)
- Web Service: $7/month (512MB RAM, always on)

**Netlify:**
- Free tier sufficient for most use cases
- Pro: $19/month (optional, for advanced features)

**Total: $14-33/month**
