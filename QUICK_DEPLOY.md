## 🚀 Deploy in 5 Steps

### 1. Deploy Backend (Render)
```
1. Go to: https://dashboard.render.com
2. Click: New + → Web Service
3. Connect: Adi6tnine/Algo-mentor
4. Click: Apply (auto-detects render.yaml)
5. Wait: 5-10 minutes
```

**Result:** `https://algomentor-backend.onrender.com`

### 2. Deploy Frontend (Netlify)
```
1. Go to: https://app.netlify.com
2. Click: Add new site → Import project
3. Connect: Adi6tnine/Algo-mentor
4. Click: Deploy (auto-detects netlify.toml)
5. Wait: 2-5 minutes
```

**Result:** `https://[random-name].netlify.app`

### 3. Update CORS
```
1. Go to: Render Dashboard → algomentor-backend
2. Click: Environment tab
3. Update: CORS_ALLOWED_ORIGINS=https://[your-netlify-url].netlify.app
4. Click: Save Changes
5. Wait: Auto-redeploy
```

### 4. Test
```
1. Visit: https://[your-netlify-url].netlify.app
2. Click: Sign Up
3. Create: Student account
4. Login: With credentials
5. Test: Dashboard, Problems, Sync
```

### 5. Done! 🎉
```
✅ Backend: Running on Render
✅ Frontend: Running on Netlify
✅ Database: PostgreSQL on Render
✅ CORS: Configured
✅ HTTPS: Enabled automatically
```

## 🐛 Quick Troubleshooting

### Backend Won't Start
```
1. Check: Render Dashboard → Logs
2. Verify: Java 17+ in render.yaml
3. Check: Database is running
4. Verify: Environment variables set
```

### Frontend Won't Load
```
1. Check: Netlify Dashboard → Deploys
2. Verify: Build succeeded
3. Check: Base directory = "frontend"
4. Verify: Publish directory = "build"
```

### CORS Errors
```
1. Update: CORS_ALLOWED_ORIGINS in Render
2. Include: Your Netlify URL
3. Save: Changes
4. Wait: Auto-redeploy
5. Clear: Browser cache
```

### Can't Login
```
1. Check: Browser console for errors
2. Verify: Backend is running
3. Check: CORS is configured
4. Verify: API URL is correct
5. Try: Different browser
```
