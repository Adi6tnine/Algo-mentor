## рџђ› Common Issues & Solutions

### Issue 1: Build Fails on Render
**Solution**: Check build logs in Render dashboard. Common causes:
- Maven dependencies not downloading
- Java version mismatch
- Out of memory (upgrade plan)

### Issue 2: Frontend Can't Connect to Backend
**Solution**: 
1. Verify backend is running on Render
2. Check `REACT_APP_API_URL` in `netlify.toml`
3. Update `CORS_ALLOWED_ORIGINS` in Render
4. Check browser console for CORS errors

### Issue 3: Database Connection Failed
**Solution**:
1. Verify PostgreSQL database is running on Render
2. Check environment variables are set correctly
3. Ensure database and web service are in same region

### Issue 4: JWT Authentication Fails
**Solution**:
1. Verify `JWT_SECRET` is set in Render
2. Check token expiration time
3. Clear browser localStorage and login again
